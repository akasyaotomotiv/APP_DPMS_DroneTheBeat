## 프로잭트 소개
- DPMS(드론 비행 허가 시스템)

## 기능 설명
 - 드론 비행을 희망하는 사람과 군내 기지방호작전과 간의 정보 공유를 편리하게 만들어 드론 비행의 승인 및 감시가 용이해짐
 - 개인 신상 및 드론 정보, 비행 반경을 미리 양식에 입력하여 군으로 전송하므로 신속하게 승인 가능
 - 비행 금지 구역에서 비행하는 사고를 미리 예방할 수 있음

## 컴퓨터 구성 / 필수 조건 안내 (Prerequisites)
* ECMAScript 6 지원 브라우저 사용
* 권장: Google Chrome 버젼 77 이상

## 기술 스택 (Technique Used) 
### Front-end
 -  Flutter

### Back-end
 -  Firebase
 

## 설치 안내 (Installation Process)
```bash
$ git clone https://github.com/osamhack2022-v2/APP_DPMS_DroneTheBeat.git
$ yarn or npm install
$ yarn start or npm run start
```
